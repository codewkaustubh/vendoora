# VENDOORA Brand Guidelines

## Brand Voice

VENDOORA should feel premium, aspirational, and trustworthy. The voice should be polished, warm, and efficient, reflecting both luxury event planning and modern marketplace convenience.

## Colors

- Deep cobalt for trust and action
- Sunset orange for urgency and highlights
- Mint for calm validation and success states
- Indigo/purple for premium dashboard energy

## Typography

- Poppins for clear UI and modern utility
- Playfair Display for editorial moments and elevated messaging
- JetBrains Mono for technical and small-detail interfaces

## Logo Usage

The current UI uses the VENDOORA wordmark prominently in the header. In future design work, the logo treatment should stay clean and high-contrast, with emphasis on clarity across mobile and desktop experiences.

## Glassmorphism

Use translucent layers, frosted panels, and soft blur to communicate premium polish without overwhelming the content.

## Icons

Icons should feel modern and approachable, with a clean outline treatment suitable for both consumer and enterprise-style contexts.

## Gradients

The brand favors soft, layered gradients rather than harsh neon effects. The gradients should support the premium editorial look while keeping the interface readable.

## UI Style

The overall UI should continue to feel:

- Luxury-oriented
- Friendly and accessible
- Contemporary and mobile-first
- Visually rich but not noisy
